What You're Looking At

▼

This project is an interactive visualization dashboard that analyzes the estimated cost of meeting Steam games' recommended hardware requirements. It focuses on CPU, GPU, and RAM for games released between 2015 and 2024.

This dashboard compares each game's release year with the estimated cost of building a PC in 2026 that can meet its recommended specifications.

By holding component prices in the present, the visualizations aim to isolate how demanding games are relative to one another depending on release year.
